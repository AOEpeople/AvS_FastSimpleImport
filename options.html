Options

You can find the configuration options at
System -> Configuration -> Services -> FastSimpleImport

General Settings

Import Behavior

Select from:

  • Append Complex Data: create new products, modify existing products
  • Replace Existing Complex Data: create new products, delete and recreate existing products
  • Delete Entities: delete given products, only "sku" column needed
